### 汉诺塔
- 什么是汉诺塔
  - 是根据一个传说形成的数学问题：

有三根杆子A，B，C。A杆上有 N 个 (N>1) 穿孔圆盘，盘的尺寸由下到上依次变小。要求按下列规则将所有圆盘移至 C 杆：

  - 1、每次只能移动一个圆盘；
  - 2、大盘不能叠在小盘上面。

提示：可将圆盘临时置于 B 杆，也可将从 A 杆移出的圆盘重新移回 A 杆，但都必须遵循上述两条规则。

盘子：d1,d2,...,dn;(下标越大，盘子就越大！)
- n=1:
  - d1 -> C;
- n=2:
  - d1 -> C d1 -> B d2 -> C d1 -> C 
- n=3:
  - d1 -> C d2 -> B d1 -> B d3 -> C
  d1 -> A d2 -> C d1 -> C
- n=4:
  - d1 -> C d2 -> B d1 -> B d3 -> C
   d1 -> C d2 -> A  d1 -> A d3 -> B
   d1 -> C d2 -> B d1 -> B d4 -> C
   d1 -> C d2 -> A d1 -> A d3 -> C
   d1 -> B d2 -> C d1 -> C
- n=5:
  - d1 -> C d2 -> B d1 -> B d3 - C
   d1 -> A d2 -> C d1 -> C d4 -> B
   d1 -> B d2 -> A d1 -> A d3 -> B
   d1 -> C d2 -> B d1 -> B d5 -> C
   d1 -> A d2 -> C d1 -> C d3 -> A
   d1 -> B d2 -> A d1 -> A d4 -> C
   d1 -> C d2 -> B d1 -> B d3 -> C
   d1 -> A d2 -> C d1 -> C

规律：
n = 1时：只要把d1移动C即可。
n = 2时：首先把d1移到B，其次把d2移到C，最后把d1移到C。
n = 3时：首先把d1和d2移到B，其次把d3移到C，最后把d1和d2移到C。
n = 4时：首先把d1、d2和d3移到B，其次把d4移到C，最后把d1、d2和d3移到C。
n = 5时：首先把d1、d2、d3和d4移到B，其次把d5移到C，最后把d1、d2、d3和d4移到C。

当n=k时：先把n-1层移到B，其次把dk移到C，最后把n-1层移到C。
